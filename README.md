### BDD Process
Add a task
* **Input:**
  * Workout on tuesday
* **Output:**
  * "Workout on tuesday"
Click on task to ask if task is completed.
* **Input:**
  * "Workout on tuesday"
* **Output:**
  * "Are you finished with this task: Y/N?"
If yes, remove task from list.
* **Input:**
  * Yes
* **Output:**
  * "Task removed"
